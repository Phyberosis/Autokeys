# Autokeys

a simple mouse and keyboard macro


enter letters for it to type

CTRL + SHIFT + CLICK to copy mouse position
CTRL + E to paste
add L or R for left/right click
ex @|!ML(480, 704)L;

CTRL + D for @|! to start command keys
; to end

@|!W500;  to wait 500ms
@|!#c;    CTRL + c
@|!^T;    ATL + TAB
